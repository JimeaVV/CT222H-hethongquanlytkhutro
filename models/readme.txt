1. Tầng Models (Entities)
Bạn cần tạo một thư mục models/ để tiến hành "bản đồ hóa" (Mapping) các bảng SQL thành các Class trong ngôn ngữ lập trình.
Nên tách mỗi bảng thành một file riêng, ví dụ: tạo class PhongTro với các thuộc tính id, ten_phong, gia_phong, trang_thai.
Việc này giúp các bạn thao tác với database bằng code (hướng đối tượng) cực kỳ nhanh thay vì phải gõ đi gõ lại các câu lệnh SQL thuần.
